Política de Privacidad

En Mahjong Scorer, la privacidad de nuestros usuarios es prioritaria. Esta aplicación es una PWA (Progressive Web App) diseñada para funcionar con el mínimo de datos personales.

1. Recopilación de Datos

No solicitamos registro ni almacenamos datos personales identificables. Los audios enviados para el arbitraje se procesan mediante la API de Google Gemini de forma efímera y no se guardan en nuestros servidores.

2. Cookies y Publicidad

Utilizamos Google AdSense para mostrar anuncios. Google utiliza cookies para mostrar anuncios basados en las visitas previas del usuario a este sitio web u otros sitios web.

3. Contacto

Para cualquier duda, puedes contactar con el desarrollador a través del enlace de soporte en la página principal.
